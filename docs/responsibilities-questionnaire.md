# Founder Responsibilities Assessment Questionnaire
## PeptideTracker Role Definition & Alignment Tool

---

## Instructions
1. Each founder completes this independently first
2. Compare answers in joint session
3. Identify gaps, overlaps, and conflicts
4. Create final RACI matrix based on results
5. Review quarterly and adjust as needed

**Scoring Guide:**
- **Primary (P)**: I own this completely (80-100% responsibility)
- **Shared (S)**: We collaborate equally (40-60% each)
- **Support (A)**: I assist when needed (20-40% responsibility)
- **Minimal (M)**: Little to no involvement (<20%)
- **None (N)**: Not my responsibility (0%)
